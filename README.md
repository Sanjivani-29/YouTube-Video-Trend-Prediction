# 📌 Project Overview

This project analyzes YouTube trending video data to identify patterns influencing video popularity and trends. It includes data preprocessing, exploratory data analysis (EDA), feature engineering, and machine learning techniques to understand and predict video performance.

The project demonstrates an end-to-end data analysis workflow using Python and popular data science libraries.

---

# 🎯 Objectives

- Analyze YouTube trending video data.
- Clean and preprocess raw data.
- Perform exploratory data analysis (EDA).
- Identify factors affecting video trends.
- Visualize important insights.
- Build predictive models using machine learning.

# 🚀 How to Run

Clone the repository

```bash
git clone https://github.com/Sanjivani-29/YouTube-Video-Trend-Prediction.git
```

Navigate to the project folder

```bash
cd YouTube-Video-Trend-Prediction
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the notebook

```bash
jupyter notebook YouTube_Video_Trend_Prediction.ipynb
```
